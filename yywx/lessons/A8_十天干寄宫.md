# A8 十天干寄宫（串宫压运使用）

目标：熟记十天干寄宫的禄位对应，理解在“串宫压运”中的应用方式，做到在盘中能够迅速定位天干所依宫位的禄根。

## 十天干寄宫（禄位）
- 甲 → 寅
- 乙 → 卯
- 丙 → 巳
- 丁 → 午
- 戊 → 戌
- 己 → 未
- 庚 → 申
- 辛 → 酉
- 壬 → 亥
- 癸 → 子

说明：寄宫即天干之禄所归，实占中可用作“寻根与取势”的快速路标；与“根基/出处”配套使用时，寄宫更偏向禄根位置的直观定位。

## 与实占的结合（串宫压运）
- 串宫：按宫位与寄宫关系，将天干力量沿宫位串联，观察在不同位（年、月、日、时）上的禄根承接与传导。
- 压运：结合岁运流转，天干所寄宫在岁运中的受压与泄通，体现为吉凶与强弱的节律变化。

使用提示：
- 寄宫用于“定位”，根基用于“强弱”，出处用于“性质”；三者合参更稳妥。
- 先定秩序（宫位优先），再看顺序（本气/余气流程），最后以寄宫校验禄根是否落点得当。

## 练习建议
- 背诵卡片：随机抽一个天干，快速说出其寄宫（禄位）。
- 盘例圈点：选一盘，标注出日主与关键干的寄宫，观察其在宫位与岁运中的表现。
