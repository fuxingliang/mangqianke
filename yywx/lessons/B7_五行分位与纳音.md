# B7 · 五行分位（正五行 / 真五行）与纳音

本课目标：

- 掌握“正五行”的基本分位（天干与地支的常见五行归属）
- 理解“真五行”的训练口径（按天干五合、地支六合的合化五行）
- 熟悉六十甲子纳音的名称与所属元素，能快速查认常见配对

提示：命理体系中对“合化”与“纳音”的使用各有流派，课程采用简化统一口径，便于基础训练与题库练习，实际应用时需结合条件（透出、通根、月令、局势等）与流派原则酌用。

---

## 正五行总览（常见分位）

天干（十干）正五行：

- 甲乙：木
- 丙丁：火
- 戊己：土
- 庚辛：金
- 壬癸：水

地支（十二支）正五行：

- 子：水；丑：土；寅：木；卯：木；辰：土；巳：火；午：火；未：土；申：金；酉：金；戌：土；亥：水

---

## 真五行总览（训练口径：按合化五行）

说明：以“合化”为依据，将干支的“真五行”取为其合化所对应的五行，便于建立记忆与操作一致性。实际命理判断中“真五行”并非所有场景都直接等同于合化结果，本课程仅用于练习与记忆训练。

天干五合（合化）：

- 甲己合土 → 甲、己 的真五行取“土”
- 乙庚合金 → 乙、庚 的真五行取“金”
- 丙辛合水 → 丙、辛 的真五行取“水”
- 丁壬合木 → 丁、壬 的真五行取“木”
- 戊癸合火 → 戊、癸 的真五行取“火”

地支六合（合化）：

- 子丑合土 → 子、丑 的真五行取“土”
- 寅亥合木 → 寅、亥 的真五行取“木”
- 卯戌合火 → 卯、戌 的真五行取“火”
- 辰酉合金 → 辰、酉 的真五行取“金”
- 巳申合水 → 巳、申 的真五行取“水”
- 午未合土 → 午、未 的真五行取“土”

## 六十甲子 · 纳音五行（名称与元素）

记忆方法：六十甲子两年一组，同归一“纳音名称”。名称多含元素字样（如“金、木、水、火、土”），亦兼寓喻象。

## 学习与练习建议

- 先背“天干正五行、地支正五行”，形成稳定分位基准
- 再记“天干五合、地支六合”的合化五行，用于本课“真五行”训练口径
- 纳音先记名称与配对，分类到元素时可先抓带字迹象（如“金、火、水、木、土”）
- 结合题库进行分层训练：先正五行 → 真五行 → 纳音名称与元素识别
