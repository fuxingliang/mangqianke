# A9 地支本意（后天八卦）

目标：熟记地支与后天八卦的对应关系及家人角色本意，用于盘中快速取象与宫位定位。

## 后天八卦对应
- 戌、亥 → 乾宫（父）
- 未、申 → 坤宫（母）
- 辰、巳 → 巽宫（长女）
- 寅、丑 → 艮宫（少男）
- 卯 → 震宫（长男）
- 子 → 坎宫（中男）
- 酉 → 兑宫（少女）
- 午 → 离宫（中女）

说明：此为地支本意的宫位映射，常用于“后天八卦取象”。与“根基/出处、寄宫、秩序/顺序”联用时，可更快锁定人物与宫位取象。

## 使用提示
- 宫位取象：先定宫位（后天八卦），再合参干支力量与性质，避免单点判断。
- 与秩序/顺序结合：秩序用来定优先宫位，顺序用来定流程，地支本意用来补充人物角色取象。

## 练习建议
- 背诵卡片：随机抽地支，说出其宫位与对应家人角色。
- 盘例圈点：给定盘面，标注地支宫位，写出人物取象与关系线索。
